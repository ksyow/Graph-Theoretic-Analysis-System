What is Planarity?

A planar graph is a graph that can be drawn on the plane without any edge crossings.

The complete graph K₅ and bipartite graph K₃,₃ are two classic examples of non-planar graphs, also known as Kuratowski graphs.

Properties of Planar Graphs

  • A planar graph can be drawn without edge crossings.
  • Connected planar graphs satisfy Euler’s formula, that is for a connected planar graph with \(V\) vertices, \(E\) edges and \(F\) faces, the graph satisfies the formula:

    \[ V - E + F = 2 \]

  • For a connected planar graph with V vertices and E edges: \[ E \leq 3V - 6 \]
  • Planar graphs divide the plane into faces.

Kuratowski's Theorem

A graph is planar if and only if it does not contain a subdivision of K₅ or K₃,₃. Symbolically, we can write this statement as:

\[ G \text{ is planar } \iff G \text{ contains no subdivision of } K_5 \text{ or } K_{3,3} \]

The theorem is significant because it indicates that a graph is not planar if we find a subdivision of K₅ or K₃,₃.

What is Connectivity in Undirected Graphs?

A graph is said to be connected if every two vertices are joined by a path.

The graph that is not connected is called a disconnected graph.

Properties of Connected Graphs

  • Any simple graph with \(n\) vertices and more than \((n - 1)(n - 2)/2\) edges is connected.
  • Connected planar graphs satisfy Euler’s formula: \(V - E + F = 2\)
  • \(E \leq 3V - 6\)

Edge Connectivity

\(\lambda(G)\): minimum edges needed to disconnect graph.

Vertex Connectivity

\(\kappa(G)\): minimum vertices needed to disconnect graph.

What is Connectivity in Directed Graphs?

There are two types: strongly connected and weakly connected.

Strongly Connected

For every \(u,v\), there exists a directed path \(u \to v\) and \(v \to u\).

Weakly Connected

If we ignore edge directions and the graph becomes connected.

What is the Tutte Polynomial?

The Tutte polynomial is a two-variable polynomial associated with a graph. It is one of the most important graph polynomials in graph theory because it contains significant combinatorial information about a graph.

The Tutte polynomial of a graph \(G\) is denoted by:

\[ T(G;x,y) \]

Many graph invariants such as spanning trees, spanning forests, connected spanning subgraphs, and chromatic-related properties can be obtained from special evaluations of the Tutte polynomial.

Deletion-Contraction Definition

The Tutte polynomial satisfies the recursive deletion-contraction relation:

\[ T(G;x,y) = T(G-e;x,y) + T(G/e;x,y) \]

  • \(G-e\) denotes the graph obtained by deleting edge \(e\).
  • \(G/e\) denotes the graph obtained by contracting edge \(e\).

This recursive definition is one of the most common methods used to compute the Tutte polynomial.

If the graph consists only of \(i\) bridges and \(j\) loops, then:

\[ T(G;x,y)=x^i y^j \]

Rank-Nullity Formula

The Tutte polynomial can also be expressed using the rank-nullity generating function:

\[ T(G;x,y) = \sum_{A\subseteq E} (x-1)^{r(E)-r(A)} (y-1)^{n(A)} \]

  • \(A \subseteq E\) means that \(A\) is a subset of edges.
  • \(r(A)\) denotes the rank of the subgraph.
  • \(n(A)\) denotes the nullity of the subgraph.

This formulation is useful in combinatorial counting and theoretical analysis.

Special Evaluations of the Tutte Polynomial

Several important graph quantities can be obtained by evaluating the Tutte polynomial at specific points.

  • \[ T(G;1,1)=\tau(G) \] equals the number of spanning trees of \(G\).
  • \[ T(G;2,1) \] equals the number of spanning forests of \(G\).
  • \[ T(G;1,2) \] equals the number of connected spanning subgraphs of \(G\).
  • \[ T(G;2,2)=2^{|E|} \] where \(|E|\) denotes the number of edges of \(G\).

Properties of the Tutte Polynomial

  • The Tutte polynomial is independent of the order in which edges are deleted or contracted.
  • The coefficients of the Tutte polynomial are non-negative integers.
  • The Tutte polynomial generalizes many important graph invariants.
  • For planar graphs, the Tutte polynomial satisfies a duality relation with the dual graph.

What is Chromatic Number?

The chromatic number of a graph \(G\), represented as \(\chi(G)\), is the smallest number of colours needed to colour its vertices so that no two adjacent vertices have the same colour.

Chromatic Numbers of Common Graphs

  • Complete Graph \(K_n\)
    Every vertex is adjacent to all other vertices, therefore: \[ \chi(K_n)=n \]

  • Cycle Graph \(C_n\)
    If the cycle length is even: \[ \chi(C_n)=2 \] If the cycle length is odd: \[ \chi(C_n)=3 \]

  • Bipartite Graph
    Any non-empty bipartite graph can be colored using two colors: \[ \chi(G)=2 \] A graph is bipartite if and only if it is 2-colorable.

  • Star Graph
    A star graph is bipartite, therefore: \[ \chi(G)=2 \]

  • Wheel Graph \(W_n\)
    If the number of vertices is even: \[ \chi(W_n)=3 \] If the number of vertices is odd: \[ \chi(W_n)=4 \]

  • Tree
    Every tree with at least one edge satisfies: \[ \chi(T)=2 \]

  • Planar Graph
    By the Four Color Theorem: \[ \chi(G)\leq 4 \]

  • Edgeless Graph
    If a graph contains no edges, then only one color is needed: \[ \chi(G)=1 \]

Properties of the Chromatic Number

  • The chromatic number is always at least the size of the largest clique in the graph.
  • A graph with chromatic number 2 is called bicolorable.
  • A graph with chromatic number 3 is called three-colorable.
  • According to Brooks’ Theorem, the chromatic number of a graph is at most the maximum vertex degree, except for complete graphs and odd cycles.
  • Determining the chromatic number of a graph is generally a difficult combinatorial optimization problem.

What is Centrality?

In graph theory, the significance of nodes in a network is assessed using centrality measures. They aid in determining whether nodes are significant information carriers, powerful, or well-connected.

Each node in a graph is given a numerical value by centrality measurements according to its significance within the network.

Centrality is widely used in social networks, transportation systems, communication networks, biology, and information analysis.

Degree Centrality

Degree centrality measures how many edges are connected to a vertex.

For a graph with \(n\) vertices:

\[ C_D(v)=\frac{\deg(v)}{n-1} \]

  • \(\deg(v)\) is the number of edges connected to vertex \(v\). We known it as degree of vertex \(v\).
  • Vertices with high degree centrality are directly connected to many vertices.
  • In directed graphs, degree centrality can be divided into:
    • In-degree centrality
    • Out-degree centrality

Betweenness Centrality

Betweenness centrality measures how often a vertex lies on shortest paths between other pairs of vertices.

\[ C_B(v)= \sum_{s \neq v \neq t} \frac{\sigma_{st}(v)}{\sigma_{st}} \]

  • \(\sigma_{st}\) is the number of shortest paths between vertices \(s\) and \(t\).
  • \(\sigma_{st}(v)\) is the number of those shortest paths passing through \(v\).
  • Vertices with high betweenness centrality often act as bridges or connectors.

Closeness Centrality

Closeness centrality measures how close a vertex is to all other vertices in the graph.

\[ C_C(v)= \frac{n-1} {\sum_{u \neq v} d(v,u)} \]

  • \(d(v,u)\) denotes the shortest-path distance between vertices \(v\) and \(u\).
  • Vertices with high closeness centrality can quickly interact with other vertices.

Properties of Centrality Measures

  • Different centrality measures may identify different important vertices.
  • Centrality values are usually normalized between 0 and 1.
  • High degree centrality does not necessarily imply high betweenness centrality.
  • Centrality measures help identify influential or critical vertices in a network.

What is a Shortest Path?

A path in a graph is a sequence of vertices connected by edges.

The shortest path between two vertices is the path with the minimum distance or cost.

Basic Terminology

  • Walk
    A sequence of vertices and edges where vertices or edges may be repeated.

  • Path
    A sequence of connected vertices with no repeated vertices.

  • Unweighted Graph
    A graph where all edges are treated equally. The shortest path is the path containing the fewest number of edges.

  • Weighted Graph
    A graph where edges are assigned numerical values such as distance, time, or cost.

  • Path Weight
    The total cost of a path obtained by summing the weights of all edges along the path.

Shortest Path Distance

The shortest path distance between vertices \(u\) and \(v\) is commonly denoted by:

\[ d(u,v) \]

where \(d(u,v)\) represents the minimum distance between the two vertices.

Properties of Shortest Paths

  • A shortest path is not necessarily unique.
  • In an unweighted graph, shortest paths minimize the number of edges.
  • In weighted graphs, shortest paths minimize the total edge weight.
  • If no path exists between two vertices, the distance is undefined or infinite.

What is Eccentricity?

The eccentricity of a vertex \(v\) is the greatest shortest-path distance from \(v\) to any other vertex in the graph.

In other words, eccentricity measures how far a vertex is from the most distant vertex reachable from it.

Mathematically:

$$ \epsilon(v)=\max_{u\in V} d(v,u) $$
  • \(d(v,u)\) is the shortest-path distance between vertices \(v\) and \(u\).
  • A vertex with small eccentricity is considered more “central” in the graph.

Related Concepts

  • Radius \((rad(G))\):
    The minimum eccentricity among all vertices in the graph. $$ rad(G)=\min_{v\in V}\epsilon(v) $$ Vertices whose eccentricity equals the radius are called the center (or central vertices).
  • Diameter \((diam(G))\):
    The maximum eccentricity among all vertices in the graph. $$ diam(G)=\max_{v\in V}\epsilon(v) $$ It represents the largest shortest-path distance in the graph.
  • Center:
    The set of vertices whose eccentricity equals the radius.
  • Periphery:
    The set of vertices whose eccentricity equals the diameter.

Important Notes

  • In a connected undirected graph, every vertex has a finite eccentricity.
  • In a disconnected graph, some vertices cannot reach others, so distances may be considered infinite.
  • For directed graphs, eccentricity is usually defined only for strongly connected graphs, where every vertex can reach every other vertex.
  • For any connected graph: $$ rad(G)\le diam(G)\le 2\,rad(G) $$
